# dotfiles

This will change the system to my preferences.

## Usage

### Installing Homebrew

Installs Homebrew and some common Homebrew formulae
```bash
./brew.sh
```

### macOS defaults
Run to customize a new Mac. (This includes my git-settings)
```bash
./macos.sh
```

### Backup
Copy settings to this repo:

```bash
./backup.sh
```
